# UniSphere

Clean initial starter version of the university collaboration platform.

## Included structure
- Django project package: `unisphere`
- App skeletons: accounts, materials, thesis, forum, jobs, complaints, events, notifications
- Global folders: templates, static, media

## Purpose
This version is intended for GitHub initial push before app-specific development starts.

## Run locally
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
